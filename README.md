# Cloth simulation

### Getting Started

- Spawn a WebGPU capable browser:

    ```shell
    google-chrome-unstable \
        --enable-features=Vulkan,UseSkiaRenderer \
        --enable-unsafe-webgpu
    ```
    
    *Tested on Google Chrome (dev) >= Version 103.0.5060.13*


- Start serving the webapp:

    ```shell
    yarn start
    ```
